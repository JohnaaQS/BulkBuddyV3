# Views

Deze map bevat de Razor views van de applicatie.

## Doel
Views tonen informatie aan de gebruiker en vormen de visuele presentatie van de applicatie.

## Wat hoort hier thuis
- Dashboard/Index.cshtml
- Account/Login.cshtml
- Shared/_Layout.cshtml

## Wat hoort hier niet thuis
- SQL
- uitgebreide berekeningen
- databasecode

## Architectuur
Deze map hoort bij de Presentation Layer.
Binnen MVC is dit de View.
