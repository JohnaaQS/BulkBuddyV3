# wwwroot

Deze map bevat statische bestanden van de applicatie.

## Doel
wwwroot wordt gebruikt voor frontendbestanden die direct door de browser geladen worden.

## Wat hoort hier thuis
- CSS
- JavaScript
- afbeeldingen
- iconen

## Wat hoort hier niet thuis
- C# businesslogica
- controllers
- repositories

## Architectuur
Deze map ondersteunt de presentatie van de applicatie, maar bevat geen server-side logica.
